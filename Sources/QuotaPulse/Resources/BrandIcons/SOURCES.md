# Brand Icon Sources

The OpenAI and Claude SVG icons in this directory come from Simple Icons.

- OpenAI: https://cdn.jsdelivr.net/npm/simple-icons@latest/icons/openai.svg
- Claude: https://cdn.jsdelivr.net/npm/simple-icons@latest/icons/claude.svg
- Simple Icons project: https://simpleicons.org/
- License file fetched from the package: https://cdn.jsdelivr.net/npm/simple-icons@latest/LICENSE.md
- Disclaimer fetched from the package: https://cdn.jsdelivr.net/npm/simple-icons@latest/DISCLAIMER.md

Simple Icons is a CC0 project, but its disclaimer notes that users should still consider brand/trademark guidelines and the individual icon data. The package metadata observed for this run listed Claude source as `https://claude.ai` and OpenAI source/guidelines as `https://openai.com/brand`.
